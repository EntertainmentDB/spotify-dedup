Spotify Dedup

Déduplicateur Spotify

Supprimez les chansons en double de vos listes de lecture et chansons enregistrées.

Rechercher et supprimer

Dedup vérifiez vos listes de lecture et vos chansons enregistrées dans votre bibliothèque Spotify. Une fois que Dedup trouve des doublons, vous pouvez les supprimer de playlist en playlist.

Sûr

Dedup supprime uniquement les chansons en double, laissant le reste de la liste de lecture et la collection de chansons enregistrées intactes.

Open source

Vous pouvez jeter un œil au code source sur GitHub. Cette application Web utilise l'API Web Spotify pour gérer les listes de lecture et les morceaux enregistrés de l'utilisateur.
